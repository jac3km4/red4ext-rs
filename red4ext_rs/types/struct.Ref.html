Struct red4ext_rs::types::Ref

source · 
pub struct Ref<T: ScriptClass>(/* private fields */);
Expand description

A strong reference to a script class. A live instance of the reference guarantees that the instance is still alive. When the reference is dropped, the strong reference count is decremented. If the strong reference count reaches zero, the underlying value is destroyed.
